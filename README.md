# Paraluman

Repository for the Paraluman bilingual publishing prototype materials.

- `web/`: Next.js + Convex + Clerk MVP implementation for Sabay Publish
- `docs/`: product and planning documents
- `assets/branding/`: logo and brand assets
- `assets/screenshots/`: reference screenshots
